# Как не потеряться

В файле how_write_articles/README.md приведен пример того, как можно писать MD-файлики и как он вообще должен выглядеть.    
Файл cucumbers/cucumbers.md является статьей примером.  
Директории выглядят не случайно - не принебрегайте папочкам, аккуратности. Далее заносим пути и информацию о всех md в файлик concepts.json.
